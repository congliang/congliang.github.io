常见漏洞

  1. 不安全的本地存储 → NSUserDefaults/Realm/CoreData 明文存储敏感信息
  2. URL Scheme 劫持 → 其他 App 注册相同 Scheme 截获回调
  3. 越狱检测可绕过 → Frida Hook 检测函数返回 false
  4. 后台截屏泄露 → 多任务切换时 App 快照可能暴露敏感信息
  5. Pasteboard 泄露 → UIPasteboard 复制内容全局可读取
